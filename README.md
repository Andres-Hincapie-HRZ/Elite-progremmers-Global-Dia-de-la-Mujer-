## 🌹 Características

- **Tarjeta 3D Interactiva**: Efecto de volteo suave con animación 3D
- **Diseño Profesional**: Colores elegantes basados en púrpura y rosa
- **Animaciones Dinámicas**: Rosas y corazones cayendo en el fondo
- **Mensaje Inspirador**: Dedicatoria especial para mujeres programadoras
- **Responsive**: Se adapta perfectamente a dispositivos móviles y desktop
- **Efectos Visuales**: Botones con hover effects y transiciones suaves

## 🚀 Demo

Abre `index.html` en tu navegador para ver la tarjeta en acción.

## 📋 Requisitos

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- No requiere instalación de dependencias
- Funciona completamente offline

## 💻 Instalación

1. Clona este repositorio:
```bash
git clone https://github.com/Andres-Hincapie-HRZ/Elite-programmers-Global-Dia-de-la-Mujer-.git
```

2. Navega al directorio:
```bash
cd Elite-programmers-Global-Dia-de-la-Mujer-
```

3. Abre el archivo `index.html` en tu navegador:
```bash
# En Windows
start index.html

# En macOS
open index.html

# En Linux
xdg-open index.html
```

## 🎨 Tecnologías Utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Animaciones, gradientes y efectos 3D
- **JavaScript**: Interactividad y animaciones dinámicas
- **Google Fonts**: Tipografía Poppins

## 🎯 Uso

1. **Frente de la tarjeta**: Muestra la imagen principal con el botón "✨ Tócame ✨"
2. **Clic en el botón**: La tarjeta gira revelando el mensaje especial
3. **Reverso de la tarjeta**: Mensaje inspirador con botón "← Volver"
4. **Animaciones de fondo**: Rosas 🌹 y corazones 💜💗 cayendo continuamente

## 🌟 Características Técnicas

### Animaciones CSS
- Efecto de volteo 3D con `transform: rotateY()`
- Animación de flotación para el ícono
- Gradientes animados en el texto
- Transiciones suaves en botones

### JavaScript Dinámico
- Generación automática de elementos cayendo
- Event listeners para interacción
- Animaciones con timing aleatorio
- Limpieza automática de elementos DOM

### Diseño Responsive
- Media queries para dispositivos móviles
- Tamaños adaptativos
- Mantiene proporciones en diferentes pantallas

## 🎨 Personalización

### Cambiar Colores
Edita las variables de color en `style.css`:
```css
/* Colores principales */
background: linear-gradient(135deg, #1a0b2e 0%, #2d1b4e 50%, #4a1942 100%);
color: #ff69b4; /* Rosa */
```

### Modificar Mensaje
Edita el contenido en `index.html`:
```html
<p>Tu mensaje personalizado aquí</p>
```

### Ajustar Animaciones
Modifica la velocidad en `script.js`:
```javascript
setInterval(createFallingElement, 600); // Cambiar intervalo
```
